## UTM пример
Лендинг автоматически перекладывает `utm_source`, `utm_medium`, `utm_campaign` из URL
в Telegram `start`-параметр (alphanumerics, ≤64 символа).

Пример: `https://landing.pages.dev/?utm_source=tgads&utm_medium=cpc&utm_campaign=clinics`
→ CTA: `https://t.me/aidirectprobot?start=tgads_landing__u_tgads_m_cpc_c_clinics`
